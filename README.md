# 📌 Project Overview

This project analyzes **8,800+ Netflix Movies and TV Shows** to uncover trends in content production, genres, ratings, release years, and global distribution. Using Python for data cleaning and exploratory data analysis (EDA), SQL for querying, Excel for preprocessing, and Power BI for interactive visualization, the project delivers meaningful insights that support data-driven business decisions.

---

# 🎯 Objectives

- Clean and preprocess raw Netflix dataset.
- Perform Exploratory Data Analysis (EDA).
- Identify trends in Movies vs TV Shows.
- Analyze genre popularity and content ratings.
- Explore country-wise content distribution.
- Build an interactive Power BI dashboard.
- Generate business insights through data visualization.

# 🔍 Key Insights

- 🎬 Movies account for nearly **70%** of Netflix's catalog.
- 📺 TV Shows represent approximately **30%** of total content.
- 🌍 The **United States** has the highest number of titles, followed by **India**.
- 🎭 Drama is one of the most popular genres.
- 📈 Netflix experienced rapid content growth after **2016**.
- 🔞 **TV-MA** is the most common maturity rating.
